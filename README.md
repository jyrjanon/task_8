## ğŸ” What is a VPN?
A Virtual Private Network (VPN) creates an encrypted tunnel between the user and the internet, hiding IP address, location, and traffic.

## ğŸ”° How a VPN Protects Privacy
- Hides your real IP address  
- Encrypts all data (AES-256)  
- Protects against network spying & tracking  
- Prevents ISP from monitoring browsing activity  

## ğŸ” VPN vs Proxy
| VPN | Proxy |
|-----|-------|
| Encrypts traffic | No encryption |
| Hides IP fully | Changes IP only |
| System-wide protection | Browser-only in most cases |
